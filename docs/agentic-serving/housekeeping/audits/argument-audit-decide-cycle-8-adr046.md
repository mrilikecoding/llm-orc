## Section 2: Framing Audit

### Question 1: What alternative framings did the evidence support?

**Alternative framing A — a slimmer, declaratively-rebuilt stabilization primitive, not a binary include-vs-replace choice.** §"Scope decision" frames the calibration subsystem's fate as "whether the Cycle-8 declarative serving product **includes** a calibration/verdict/cross-layer-signal/drift-audit subsystem or **replaces** it with seat contracts + dynamic dispatch + guards + engine-level events." This is a two-option framing. A third option the evidence supports but the ADR doesn't examine: keep AS-5's actual cross-session trust-accumulation mechanism (P1-3 above) as a small, purpose-built declarative primitive — e.g., a `stabilize:` node or Plexus-backed quality-signal accumulator — while still dropping the imperative Proceed/Reflect/Abstain machinery, the AUQ/HTC trajectory features, and the L0→L1 signal channel (all genuinely imperative-orchestration-shaped, AS-11-forbidden mechanism). The binary framing makes "replace everything" look like the only alternative to "keep the whole Cycle 4–6 apparatus," when the apparatus bundles a forbidden-shape mechanism (Proceed/Reflect/Abstain routing logic) with a still-needed capability (AS-5's stabilization signal) that nothing else in ADR-046 proposes to supply.

**Alternative framing B — the [confirm] fast-track presupposes an "orchestrator" actor that §1–§2's architecture may have already dissolved.** Six [confirm] rows — ADR-003's closed five-tool surface, ADR-004's context-accumulation summarization mandate, ADR-005's per-ReAct-iteration budget checks, ADR-006's `compose_ensemble` runtime composition, ADR-008's tool-mediated autonomy gating (and, by extension, ADR-011's "orchestrator is a model profile") — describe contracts belonging to *the orchestrator agent*: a persistent, tool-calling LLM actor (domain-model.md AS-1: "the orchestrator agent's tool-mediated ensemble invocations"; AS-6: "the orchestrator agent can compose new ensembles... The composable surface is fixed; only the arrangement is dynamic").
