# Temperature Conversion CLI Tool

## Installation
Install the package using pip:
```bash
pip install temperature-converter
```

## Usage
Run the CLI tool with the following syntax:
```bash
temperature-converter [from_unit] [to_unit] [value]
```

## Available Units
- `celsius` (°C)
- `fahrenheit` (°F)
- `kelvin` (K)

## Examples
```bash
temperature-converter celsius fahrenheit 25
temperature-converter fahrenheit celsius 98.6
temperature-converter celsius kelvin 0
```
