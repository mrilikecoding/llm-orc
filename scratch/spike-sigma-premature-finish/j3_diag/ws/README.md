b'# Temperature Converter CLI\n\nA command-line tool for converting temperatures between Celsius, Fahrenheit, and Kelvin.\n\n## Installation\n\nEnsure Python 3 is installed. The tool can be run directly from the `converters.py` module.\n\n## Usage\n\nThe tool provides the following commands:\n\n- `celsius_to_fahrenheit <value>`: Convert Celsius to Fahrenheit.\n- `fahrenheit_to_celsius <value>`: Convert Fahrenheit to Celsius.\n- `celsius_to_kelvin <value>`: Convert Celsius to Kelvin.\n\n## Examples\n\n```bash\n$ python converters.py celsius_to_fahrenheit 0\n32.0\n\n$ python converters.py fahrenheit_to_celsius 212\n100.0\n\n$ python converters.py celsius_to_kelvin -273.15\n0.0\n```\n\n## Error Handling\n\nThe tool validates inputs:\n- Non-numeric values (e.g., strings, `None`) raise `TypeError: Input must be a number`.\n- Boolean values (e.g., `True`, `False`) raise `TypeError: Booleans are not valid temperature values`.\n\n## Testing\n\nRun unit tests with `pytest`:\n\n```bash\n$ pytest test_converters.py\n```'
