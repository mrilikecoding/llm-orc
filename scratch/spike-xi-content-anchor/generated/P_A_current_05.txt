# Temperature Conversion Library

## Installation

Install the library using pip:

```bash
pip install temperature-converter
```

## Functions

The `converters` module provides the following functions for temperature conversion:

- `convert_c_to_f(temperature: float) -> float`: Converts Celsius to Fahrenheit.
- `convert_f_to_c(temperature: float) -> float`: Converts Fahrenheit to Celsius.
- `convert_k_to_c(temperature: float) -> float`: Converts Kelvin to Celsius.
- `convert_c_to_k(temperature: float) -> float`: Converts Celsius to Kelvin.
- `convert_f_to_k(temperature: float) -> float`: Converts Fahrenheit to Kelvin.
- `convert_k_to_f(temperature: float) -> float`: Converts Kelvin to Fahrenheit.

All functions take a single float argument representing the temperature and return the converted value as a float.
