# Temperature Conversion Library

## Installation
Install the library using pip:

```bash
pip install temperature-converter
```

## Usage
Import the `converters` module and use its functions to convert temperatures between Celsius, Fahrenheit, and Kelvin.

```python
from temperature_converter.converters import (
    celsius_to_fahrenheit,
    fahrenheit_to_celsius,
    celsius_to_kelvin
)
```

## Functions
The `converters` module provides these functions:

- `celsius_to_fahrenheit(celsius: float) -> float`: Converts Celsius to Fahrenheit
- `fahrenheit_to_celsius(fahrenheit: float) -> float`: Converts Fahrenheit to Celsius
- `celsius_to_kelvin(celsius: float) -> float`: Converts Celsius to Kelvin

## Example
```python
# Convert 25°C to Fahrenheit
fahrenheit = celsius_to_fahrenheit(25.0)
print(f"25°C = {fahrenheit:.2f}°F")  # 25°C = 77.00°F

# Convert 72°F to Celsius
celsius = fahrenheit_to_celsius(72.0)
print(f"72°F = {celsius:.2f}°C")  # 72°F = 22.22°C

# Convert 0°C to Kelvin
kelvin = celsius_to_kelvin(0.0)
print(f"0°C = {kelvin:.2f}K")  # 0°C = 273.15K
```
