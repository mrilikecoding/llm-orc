# Temperature Conversion Library

## Installation
Install via pip:
```bash
pip install temperature-converter
```

## Functions
- `celsius_to_fahrenheit(celsius: float) -> float`: Converts Celsius to Fahrenheit using the formula °F = °C × 9/5 + 32
- `fahrenheit_to_celsius(fahrenheit: float) -> float`: Converts Fahrenheit to Celsius using the formula °C = (°F - 32) × 5/9
- `celsius_to_kelvin(celsius: float) -> float`: Converts Celsius to Kelvin by adding 273.15

## Examples
```python
import converters

# Convert 25°C to Fahrenheit
fahrenheit = converters.celsius_to_fahrenheit(25)
print(fahrenheit)  # Output: 77.0

# Convert 77°F to Celsius
celsius = converters.fahrenheit_to_celsius(77)
print(celsius)  # Output: 25.0

# Convert 25°C to Kelvin
kelvin = converters.celsius_to_kelvin(25)
print(kelvin)  # Output: 298.15
```
