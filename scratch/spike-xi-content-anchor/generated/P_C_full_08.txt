# Temperature Conversion Library

## Installation
Install the library using pip:
```bash
pip install path/to/library
```

## Functions
- `celsius_to_fahrenheit(celsius: float) -> float`: Convert Celsius to Fahrenheit.
- `fahrenheit_to_celsius(fahrenheit: float) -> float`: Convert Fahrenheit to Celsius.
- `celsius_to_kelvin(celsius: float) -> float`: Convert Celsius to Kelvin.

## Example Usage
```python
import converters

# Convert 25°C to Fahrenheit
fahrenheit = converters.celsius_to_fahrenheit(25)
print(fahrenheit)  # Output: 77.0

# Convert 86°F to Celsius
celsius = converters.fahrenheit_to_celsius(86)
print(celsius)  # Output: 30.0

# Convert 0°C to Kelvin
kelvin = converters.celsius_to_kelvin(0)
print(kelvin)  # Output: 273.15
```
