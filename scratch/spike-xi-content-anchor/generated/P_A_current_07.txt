# Temperature Conversion Library

A lightweight Python library for converting temperatures between Celsius, Fahrenheit, and Kelvin.

## Installation

```bash
pip install temperature-converter
```

## Available Functions

The `converters` module provides the following conversion functions:

### Celsius ⇄ Fahrenheit
```python
converters.c_to_f(temperature: float) -> float
converters.f_to_c(temperature: float) -> float
```

### Celsius ⇄ Kelvin
```python
converters.c_to_k(temperature: float) -> float
converters.k_to_c(temperature: float) -> float
```

### Fahrenheit ⇄ Kelvin
```python
converters.f_to_k(temperature: float) -> float
converters.k_to_f(temperature: float) -> float
```

## Example Usage

```python
from converters import c_to_f, k_to_c

# Celsius to Fahrenheit
print(c_to_f(25.0))  # Output: 77.0

# Kelvin to Celsius
print(k_to_c(300.0))  # Output: 26.85
```

All functions accept and return `float` values. Input validation is handled internally.
