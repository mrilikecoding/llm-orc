# Temperature Conversion Library

A lightweight library for converting temperatures between Celsius, Fahrenheit, and Kelvin.

## Installation
```bash
pip install temperature-conversion
```

## Usage
Import the `converters` module and use its functions to convert temperatures.

## Functions
The `converters` module provides the following functions:

### `celsius_to_fahrenheit(celsius)`
Converts Celsius to Fahrenheit.
```python
converters.celsius_to_fahrenheit(25)
# Output: 77.0
```

### `fahrenheit_to_celsius(fahrenheit)`
Converts Fahrenheit to Celsius.
```python
converters.fahrenheit_to_celsius(77)
# Output: 25.0
```

### `celsius_to_kelvin(celsius)`
Converts Celsius to Kelvin.
```python
converts.celsius_to_kelvin(0)
# Output: 273.15
```

### `kelvin_to_celsius(kelvin)`
Converts Kelvin to Celsius.
```python
converters.kelvin_to_celsius(273.15)
# Output: 0.0
```

### `fahrenheit_to_kelvin(fahrenheit)`
Converts Fahrenheit to Kelvin.
```python
converters.fahrenheit_to_kelvin(32)
# Output: 273.15
```

### `kelvin_to_fahrenheit(kelvin)`
Converts Kelvin to Fahrenheit.
```python
converters.kelvin_to_fahrenheit(273.15)
# Output: 32.0
```

## Example Usage
```python
import converters

# Convert 100°C to Fahrenheit
print(converters.celsius_to_fahrenheit(100))  # Output: 212.0

# Convert 212°F to Celsius
print(converters.fahrenheit_to_celsius(212))  # Output: 100.0

# Convert 0°C to Kelvin
print(converters.celsius_to_kelvin(0))        # Output: 273.15
```
