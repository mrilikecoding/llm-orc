# Temperature Conversion Library

## Installation
Install the library using pip:

```bash
pip install temperature-converter
```

## Functions
The `converters` module provides the following functions:

```python
def celsius_to_fahrenheit(celsius: float) -> float
def fahrenheit_to_celsius(fahrenheit: float) -> float
def celsius_to_kelvin(celsius: float) -> float
```

### Function Details
- `celsius_to_fahrenheit`: Converts degrees Celsius to Fahrenheit.
- `fahrenheit_to_celsius`: Converts degrees Fahrenheit to Celsius.
- `celsius_to_kelvin`: Converts degrees Celsius to Kelvin.

## Example Usage
```python
import converters

# Convert 25°C to Fahrenheit
fahrenheit = converters.celsius_to_fahrenheit(25)
print(fahrenheit)  # Output: 77.0

# Convert 72°F to Celsius
celsius = converters.fahrenheit_to_celsius(72)
print(celsius)  # Output: 22.22222222222222

# Convert 0°C to Kelvin
kelvin = converters.celsius_to_kelvin(0)
print(kelvin)  # Output: 273.15
```
