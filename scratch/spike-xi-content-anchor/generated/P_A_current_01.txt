# Temperature Conversion Library

A simple library for converting temperatures between Celsius, Fahrenheit, and Kelvin.

## Installation

Install the library using pip:

```bash
pip install tempconv
```

This library is compatible with Python 3.6 and above.

## Usage

Import the `converters` module and use its functions to convert temperatures:

```python
import converters

# Convert Celsius to Fahrenheit
print(converters.c_to_f(25))  # Output: 77.0

# Convert Fahrenheit to Celsius
print(converters.f_to_c(77))  # Output: 25.0

# Convert Kelvin to Celsius
print(converters.k_to_c(300))  # Output: 26.85
```

## Functions

The `converters` module provides the following functions:

- `c_to_f(temperature: float) -> float`: Converts Celsius to Fahrenheit.
- `f_to_c(temperature: float) -> float`: Converts Fahrenheit to Celsius.
- `k_to_c(temperature: float) -> float`: Converts Kelvin to Celsius.
- `c_to_k(temperature: float) -> float`: Converts Celsius to Kelvin.
- `f_to_k(temperature: float) -> float`: Converts Fahrenheit to Kelvin.
- `k_to_f(temperature: float) -> float`: Converts Kelvin to Fahrenheit.

All functions take a single temperature value as input and return the converted value as a float.
