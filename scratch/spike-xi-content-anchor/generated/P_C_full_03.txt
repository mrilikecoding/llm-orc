# Temperature Conversion Library  

## Installation  
Install the library via pip:  
```bash  
pip install temperature-conversion  
```  

Or, if using the source code directly:  
```bash  
pip install -e .  
```  

## Functions  
The `converters` module provides the following functions:  

### `celsius_to_fahrenheit(celsius: float) -> float`  
Converts Celsius to Fahrenheit.  
```python  
converters.celsius_to_fahrenheit(25)  # Returns 77.0  
```  

### `fahrenheit_to_celsius(fahrenheit: float) -> float`  
Converts Fahrenheit to Celsius.  
```python  
converters.fahrenheit_to_celsius(77)  # Returns 25.0  
```  

### `celsius_to_kelvin(celsius: float) -> float`  
Converts Celsius to Kelvin.  
```python  
converters.celsius_to_kelvin(0)  # Returns 273.15  
```  

## Example Usage  
```python  
import converters  

# Convert 25°C to Fahrenheit  
print(converters.celsius_to_fahrenheit(25))  # Output: 77.0  

# Convert 77°F to Celsius  
print(converters.fahrenheit_to_celsius(77))  # Output: 25.0  

# Convert 0°C to Kelvin  
print(converters.celsius_to_kelvin(0))  # Output: 273.15  
```
