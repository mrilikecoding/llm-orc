# Temperature Conversion Library

A simple library for converting temperatures between Celsius, Fahrenheit, and Kelvin.

## Installation

Install via pip:

```bash
pip install temperature-converter
```

## Usage

Import the `converters` module and use the provided functions:

```python
import converters

# Example conversions
print(converters.celsius_to_fahrenheit(25))        # Output: 77.0
print(converters.fahrenheit_to_celsius(77))        # Output: 25.0
print(converters.celsius_to_kelvin(0))             # Output: 273.15
```

## Functions

- `celsius_to_fahrenheit(celsius: float) -> float`
  Converts a temperature from Celsius to Fahrenheit.
- `fahrenheit_to_celsius(fahrenheit: float) -> float`
  Converts a temperature from Fahrenheit to Celsius.
- `celsius_to_kelvin(celsius: float) -> float`
  Converts a temperature from Celsius to Kelvin.
