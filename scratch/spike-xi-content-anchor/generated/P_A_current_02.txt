# Temperature Conversion Library

A lightweight library for converting temperatures between Celsius, Fahrenheit, and Kelvin.

## Installation

Install via pip:

```bash
pip install temperature-converter
```

## Functions

The `converters` module provides the following functions:

- `celsius_to_fahrenheit(degrees: float) -> float`: Converts Celsius to Fahrenheit.
- `fahrenheit_to_celsius(degrees: float) -> float`: Converts Fahrenheit to Celsius.
- `kelvin_to_celsius(degrees: float) -> float`: Converts Kelvin to Celsius.
- `celsius_to_kelvin(degrees: float) -> float`: Converts Celsius to Kelvin.

## Example Usage

```python
from converters import celsius_to_fahrenheit, kelvin_to_celsius

# Convert 25°C to Fahrenheit
print(celsius_to_fahrenheit(25))  # Output: 77.0

# Convert 300 K to Celsius
print(kelvin_to_celsius(300))     # Output: 26.85
```
