# Temperature Conversion Library

## Installation
Install via pip:
```bash
pip install your_username/temperature-converter
```

Or download the source and run:
```bash
python setup.py install
```

## Functions
The `converters` module provides these temperature conversion functions:

- `celsius_to_fahrenheit(celsius: float) -> float`: Convert Celsius to Fahrenheit
- `fahrenheit_to_celsius(fahrenheit: float) -> float`: Convert Fahrenheit to Celsius
- `celsius_to_kelvin(celsius: float) -> float`: Convert Celsius to Kelvin

## Example Usage
```python
from converters import celsius_to_fahrenheit, fahrenheit_to_celsius, celsius_to_kelvin

# Convert 25°C to Fahrenheit
fahrenheit = celsius_to_fahrenheit(25)
print(f"25°C = {fahrenheit}°F")  # Output: 25°C = 77.0°F

# Convert 77°F back to Celsius
celsius = fahrenheit_to_celsius(77)
print(f"77°F = {celsius}°C")  # Output: 77°F = 25.0°C

# Convert 25°C to Kelvin
kelvin = celsius_to_kelvin(25)
print(f"25°C = {kelvin}K")  # Output: 25°C = 298.15K
```
