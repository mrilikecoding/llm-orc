# Temperature Conversion Library

## Installation  
Install the library using pip:  
```bash
pip install temperature-converter
```

## Functions  
The `converters` module provides the following functions:  

### `celsius_to_fahrenheit(celsius: float) -> float`  
Converts Celsius to Fahrenheit.  
**Example**:  
```python
converters.celsius_to_fahrenheit(25.0)
```

### `fahrenheit_to_celsius(fahrenheit: float) -> float`  
Converts Fahrenheit to Celsius.  
**Example**:  
```python
converters.fahrenheit_to_celsius(77.0)
```

### `celsius_to_kelvin(celsius: float) -> float`  
Converts Celsius to Kelvin.  
**Example**:  
```python
converters.celsius_to_kelvin(0.0)
```

## Example Usage  
```python
import converters

# Convert 25°C to Fahrenheit
fahrenheit = converters.celsius_to_fahrenheit(25.0)
print(f"25°C = {fahrenheit}°F")  # Output: 25°C = 77.0°F

# Convert 77°F to Celsius
celsius = converters.fahrenheit_to_celsius(77.0)
print(f"77°F = {celsius}°C")  # Output: 77°F = 25.0°C

# Convert 0°C to Kelvin
kelvin = converters.celsius_to_kelvin(0.0)
print(f"0°C = {kelvin}K")  # Output: 0°C = 273.15K
```
