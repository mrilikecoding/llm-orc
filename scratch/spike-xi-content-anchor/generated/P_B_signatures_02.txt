# Temperature Conversion Library

## Installation

Install via pip:
```bash
pip install temperature-converter
```

The package is available on PyPI and provides a single module `converters` with temperature conversion functions.

## API Reference

The `converters` module provides the following functions:

### `celsius_to_fahrenheit(celsius: float) -> float`
Converts Celsius to Fahrenheit using the formula:  
°F = °C × 9/5 + 32

### `fahrenheit_to_celsius(fahrenheit: float) -> float`
Converts Fahrenheit to Celsius using the formula:  
°C = (°F - 32) × 5/9

### `celsius_to_kelvin(celsius: float) -> float`
Converts Celsius to Kelvin using the formula:  
K = °C + 273.15

## Example Usage

```python
from converters import celsius_to_fahrenheit, fahrenheit_to_celsius, celsius_to_kelvin

# Celsius to Fahrenheit
temp_f = celsius_to_fahrenheit(25.0)
print(f"25°C = {temp_f:.2f}°F")  # 25°C = 77.00°F

# Fahrenheit to Celsius
temp_c = fahrenheit_to_celsius(77.0)
print(f"77°F = {temp_c:.2f}°C")  # 77°F = 25.00°C

# Celsius to Kelvin
temp_k = celsius_to_kelvin(0.0)
print(f"0°C = {temp_k:.2f}K")  # 0°C = 273.15K
```
