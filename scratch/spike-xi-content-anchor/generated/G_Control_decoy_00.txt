Here's a focused implementation of `scheduler.py` that reads the three specified settings directly from `settings.json`:

```python
import json

# Load settings from file
with open('settings.json') as f:
    config = json.load(f)

# Directly access the three required configuration keys
backoff_ms = config['backoff_ms']
max_queue_depth = config['max_queue_depth']
affinity_salt = config['affinity_salt']
```

This implementation:
- Uses direct subscript access for the three required keys
- Does not iterate through or process any other keys
- Assumes `settings.json` is in the same directory as the script
- Preserves the values as-is (e.g., `affinity_salt` remains a string)
